# Prosave – Smarter Real Estate Analytics for the Modern Market

**Prosave** is a clean, responsive, and lightning-fast landing page built for a cutting-edge real estate analytics platform.  
Designed to showcase property insights and data-driven decisions, Prosave combines elegant UI, fluid motion, and multilingual accessibility — all optimized for both desktop and mobile experiences.

## ✨ Features

- 🏠 **Real Estate Analytics Focus** – Tailored for showcasing smart property data tools
- 💡 **Fully Responsive** – Optimized for all devices, from large monitors to mobile screens
- 🌗 **Dual Theme Support** – Seamless switch between light and dark modes
- 🎞️ **Smooth Scroll Animations** – Built with Framer Motion for engaging transitions
- 🌍 **Internationalization (i18n)** – Powered by `next-intl` for multi-language support
- ⚡ **High Performance** – Blazing-fast load times with Next.js optimization
- 🎨 **Modern UI/UX** – Minimal, data-centric layout that feels premium and professional

## 🛠️ Tech Stack

### Frontend

- **Next.js** (App Router)
- **React**
- **Tailwind CSS**
- **Framer Motion** (Scroll + element animations)
- **next-intl** (Internationalization)

## 🧠 Key Highlights

- Built as a **landing page** for a real estate analytics SaaS platform
- Integrated **dark/light themes** for accessibility and modern aesthetics
- **Framer Motion** animations for scroll reveals and smooth transitions
- **Internationalization-ready** using `next-intl`, allowing instant locale switching
- Fully responsive grid layout for **mobile-first** design
